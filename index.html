Sobre o Projeto

O projeto "Salesforce Acessível" tem como objetivo tornar a plataforma Salesforce mais acessível para todos os usuários, incluindo aqueles com deficiências. Nossa missão é fornecer funcionalidades completas e uma experiência inclusiva.
